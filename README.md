# Code-Dojo ⚔️

Code-Dojo is a Full Stack Coding Battle Platform built with MERN stack.

## 🚀 Tech Stack

### Frontend
- React
- Tailwind CSS
- Context API

### Backend
- Node.js
- Express.js
- MongoDB
- Redis
- Socket.io

## 📁 Project Structure


## ⚡ Features

- Authentication (JWT)
- Survival Coding Mode
- Real-time Battle Mode
- Code Execution Engine
- Matchmaking System

## 🛠 Setup

### Backend

cd code-dojo-backend
npm install
npm start

### Frontend

cd code-dojo-frontend
npm install
npm start
